# Problem Statement
Design and implement a console based Tic Tac Toe game using C++ that allows two players to play on a 3×3 grid. The program should handle player turns, validate inputs, update the game board after each move, and accurately determine win or draw conditions.

# Description 
A console based Tic Tac Toe game developed in C++. It supports two player turn based gameplay on a 3×3 grid, validates inputs, updates the board after each move, and detects win or draw conditions. The project focuses on clean logic using arrays, functions, and control structures.

# Features
-Two player interactive gameplay  
-Real time board updates  
-Input validation for valid moves  
-Automatic win and draw detection  
-Simple and user friendly console interface
